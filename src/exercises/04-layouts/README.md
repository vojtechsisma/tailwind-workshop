# 04-layouts

Create simple layout described in the **[designs](https://www.figma.com/file/Z1VIqjmxV0wAdVLcVIhvQe/Tailwind-CSS-workshop?type=design&node-id=1-97&mode=design&t=qVE6HsYMMPJOjkgA-4)**, you will need to use some basic html tags such as ` <div>`, `<h3>`, `<p>`, `<img>`. The first layout should be created using CSS Flexbox and the second using grid. Asset images can be found in `assets/img` folder. Then you can adapt the layouts to smaller screens using media queries.
